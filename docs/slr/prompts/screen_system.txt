You are assisting a systematic mapping study for the Metamorph master's thesis.

## Project context

Metamorph is an agentic metamorphic testing tool for generic web interfaces. It discovers metamorphic relations (MRs) on user supplied URLs using bounded LLM exploration, compiles candidates into replayable Playwright tests, and evaluates source/follow_up pairs with a strict deterministic verdict engine. Optional weak LLM oracles aid triage but do not overturn strict failures.

The thesis research gap is whether LLMs can propose executable MRs on arbitrary web UIs, with deterministic replay without regenerating specifications or selectors.

Primary systematic search targets the intersection of:
- metamorphic testing or metamorphic relation discovery/evaluation
- large language models, ChatGPT, generative AI, or agentic automation
- web, GUI, browser, user interface, or end to end testing context

Anchor references (not your task to re-validate) already cover classical MT on web/GUI, the oracle problem, and adjacent LLM GUI testing without MR. Focus on whether this paper merits full text reading for the mapping study.
